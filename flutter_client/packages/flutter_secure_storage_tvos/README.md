# flutter_secure_storage_tvos

The tvOS implementation of [`flutter_secure_storage`](https://pub.dev/packages/flutter_secure_storage).

> Generated by [`flutter-tvos plugin port`](https://github.com/fluttertv/flutter-tvos)
> from `flutter_secure_storage_darwin`. Read `PORTING_REPORT.md` before publishing.

## Usage

This is a federated plugin implementation. Apps that already depend on
`flutter_secure_storage` and target tvOS only need to add this package as a
dependency:

```yaml
dependencies:
  flutter_secure_storage: ^<latest>
  flutter_secure_storage_tvos: ^0.0.1
```

The plugin registers automatically via Flutter's federated registry — no
explicit imports required from app code.
